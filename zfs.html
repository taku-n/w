ZFS of FreeBSD

FreeBSD は、GEOM (GEOMetry) によってパーティションを管理している?
ので、パーティションの状況を見るには、つぎのコマンドを使う

$ gpart show

ZFS ストレージプールの状況を出力する

$ zpool list

ZFS ファイルシステムの状況を出力する

$ zfs list

ZFS ファイルシステムを作成する

$ sudo zfs create -o mountpoint=/usr/home/hoge zroot/usr/home/hoge

ZFS ファイルシステムのクオータを 1GB に設定する

$ sudo zfs set quota=1G zroot/usr/home/hoge

ZFS ファイルシステムを削除する

$ sudo zfs destroy zroot/usr/home/hoge
