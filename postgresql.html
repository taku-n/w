とりあえず UNIX 上のユーザ hoge から PostgreSQL にログインできるようにする

ロールを作成

CREATE ROLE hoge WITH SUPERUSER LOGIN PASSWORD 'qwerty';

データベースを作成

CREATE DATABASE hoge;

PostgreSQL を抜ける

\q

UNIX ユーザ hoge のシェルから PostgreSQL にログイン

$ psql

\du
