tmux

FreeBSD では、コピーモードが動かない
GNOME 端末で、マウスで選択したり、Ctrl を押しながら選択したりして代用

設定ファイル

~/.tmux.conf

設定 そのまえに

set-option の省略形が set
set-window-option の省略形が setw
bind-key の省略形が bind

prefix key を Ctrl+V にする

set -g prefix C-v

vi での Esc の効きを快適に

set -s escape-time 0

コピーモードの動作を vi 風に

setw -g mode-keys vi

矩形でコピーするには

コピーモードで V キーを押す
もう一度押すと、非矩形に戻る

コピーモードの注意点

タブをコピーすると、コピー先ではスペースになってしまう

色の設定

set-option -g default-terminal screen-256color
set -g terminal-overrides 'xterm:colors=256'

hjkl でウインドウの移動

bind-key h select-pane -L
bind-key j select-pane -D
bind-key k select-pane -U
bind-key l select-pane -R
