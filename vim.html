set と setlocal (要検証)

set

1. File A を編集 (indent 4columns)
2. File B を編集
3. set でインデントを 2columns にする
4. File A も 2columns になってしまうはず

setlocal

1. File A を編集 (indent 4columns)
2. File B を編集
3. setlocal でインデントを 2columns にする
4. File A は 2columns のまま

Vim 全体の設定を変更するか
